# Run Islam AI (Windows / VS Code)

## 1) Open in VS Code
Unzip the folder, then open the **islam-ai** folder in VS Code.

## 2) Install dependencies
Open VS Code Terminal (PowerShell) and run:

```powershell
npm install
```

## 3) Start the dev server
```powershell
npm run dev
```

Open:
- http://localhost:3000

See **NETLIFY_DEPLOY.md**.
